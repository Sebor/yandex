Легенда

Механизм cgroups позволяет оценивать процессорное время, потраченное группой процессов. Во входном файле input.tgz находится архив содержимого каталога /sys/fs/cgroup, созданный в системе Ubuntu 12.04 командой

sudo sh -c 'mkdir -p /tmp/sys/fs ; cd /tmp; cp -r /sys/fs/cgroup /tmp/sys/fs/; tar czf input.tgz sys/'

В этой системе в момент создания архива исполнялись несколько контейнеров lxc. Выведите в файл output.txt или на стандартный вывод имя контейнера, израсходовавшего наибольшее количество процессорного времени (по сравнению с другими контейнерами lxc).
Формат ввода

Архив input.tgz в формате tar.gz
Формат вывода

Строка - имя контейнера lxc, полученная на основе анализа входного файла.
Примечания

Ненулевой код завершения система воспринимает как сообщение о неполадке, при корректном завершении программы обязательно возвращать 0
